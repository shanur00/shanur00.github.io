Flutter Notes: Image Widget

1. Image Sources: Network, Asset, and Providers

Flutter offers different constructors and classes depending on where your image is coming from and how it is being used.

Important Rules:
  • Image.network: A widget used to display an image fetched from the internet. Can include a loadingBuilder to show a progress indicator while downloading.
  • Image.asset: A widget used to display local images bundled with the app. You must declare the asset path in pubspec.yaml. Flutter automatically serves high-resolution versions if you organize them into 2.0x/ and 3.0x/ folders.
  • NetworkImage vs Image.network: Image.network is the actual UI widget. NetworkImage is an ImageProvider (raw data fetcher) used inside properties like BoxDecoration for background images.
// 1. Image.network (The UI Widget)
Image.network(
  'https://raw.githubusercontent.com/shanur00/shanur00.github.io/main/flutter/img.jpeg',
  loadingBuilder: (context, child, progress) {
    if (progress == null) return child;
    return const Center(child: CircularProgressIndicator());
  },
)

// 2. NetworkImage (The ImageProvider used for backgrounds)
Container(
  decoration: BoxDecoration(
    image: DecorationImage(
      image: NetworkImage('https://raw.githubusercontent.com/shanur00/shanur00.github.io/main/flutter/img.jpeg'),
    ),
  ),
)

// 3. Image.asset (Local File)
Image.asset(
  'assets/img.jpeg',
  errorBuilder: (context, error, stackTrace) {
    return const Center(child: Text('Asset not found!'));
  },
)

2. BoxFit Variations

The fit property determines how an image behaves when its container has a different shape or size than the original image.

Important Rules:
  • BoxFit.contain: (Default) Shrinks the image to fit entirely inside the box. Leaves empty space if proportions do not match.
  • BoxFit.cover: Scales the image to completely fill the box. Crops (chops off) parts of the image that overflow. Maintains aspect ratio.
  • BoxFit.fill: Stretches and squishes the image to fill the exact dimensions of the box. Causes visible distortion.
Container(
  width: 300,
  height: 100, // A wide, short box
  color: Colors.red[200], // Background color to reveal empty space
  child: Image.network(
    'https://raw.githubusercontent.com/shanur00/shanur00.github.io/main/flutter/img.jpeg',
    fit: BoxFit.cover, // Change to contain, cover, or fill
  ),
)

3. Color and BlendMode

These two properties work together. By themselves, images are just normal pictures. But if you apply a color, you can tint or filter the image. The colorBlendMode tells Flutter how to mix that color with the image pixels.

Common Variations:
  • BlendMode.color: Applies the color as a solid tint over the image while keeping the dark and light details.
  • BlendMode.darken: Darkens the image using the provided color. Great for making background images dark so white text is readable on top.
  • BlendMode.colorBurn: Creates a high-contrast, intense color filter effect.
Image.network(
  'https://raw.githubusercontent.com/shanur00/shanur00.github.io/main/flutter/img.jpeg',
  color: Colors.blue, // The color to mix
  colorBlendMode: BlendMode.color, // How to mix it (makes the whole image look blue)
)

4. Clipping (Rounded Corners & Circles)

The Image widget does not have a built-in property for rounded corners. You must wrap it in a clipping widget.

Important Rules:
  • ClipRRect: Clips the child widget into a Rounded Rectangle. Requires a BorderRadius.
  • ClipOval: Clips the child widget into a perfect circle or oval.
  • For clipping to work properly on images, you usually need to set the image fit to BoxFit.cover so it fills the clipped bounds.
ClipRRect(
  borderRadius: BorderRadius.circular(20),
  child: Image.network(
    'https://raw.githubusercontent.com/shanur00/shanur00.github.io/main/flutter/img.jpeg',
    width: 150,
    height: 150,
    fit: BoxFit.cover,
  ),
)

5. Full Source Code

The complete main.dart implementation demonstrating all Image concepts together, using the requested github image url.

import 'package:flutter/material.dart';

void main() {
  runApp(const MaterialApp(home: MainApp()));
}

class MainApp extends StatelessWidget {
  const MainApp({super.key});

  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(title: const Text('BoxFit Examples')),
      body: Center(
        child: Column(
          mainAxisAlignment: MainAxisAlignment.center,
          children: [
            const Text('BoxFit.contain (Leaves empty space)'),
            // I wrapped the image in a Container with a RED background.
            // This lets you see the exact shape of the box we are forcing the image into.
            Container(
              width: 300,
              height: 100, // A wide, short box
              color: Colors.red[200],
              child: Image.network(
                'https://flutter.github.io/assets-for-api-docs/assets/widgets/owl.jpg',
                fit: BoxFit.contain,
                loadingBuilder: (context, child, progress) {
                  if (progress == null) return child;
                  return const Center(child: CircularProgressIndicator());
                },
              ),
            ),

            const SizedBox(height: 30),

            const Text('BoxFit.cover (Fills box, crops top/bottom)'),
            Container(
              width: 300,
              height: 100,
              color: Colors.red[200],
              child: Image.network(
                'https://flutter.github.io/assets-for-api-docs/assets/widgets/owl.jpg',
                fit: BoxFit.cover,
              ),
            ),

            const SizedBox(height: 30),

            const Text('BoxFit.fill (Stretches and squishes)'),
            Container(
              width: 300,
              height: 100,
              color: Colors.red[200],
              child: Image.network(
                'https://flutter.github.io/assets-for-api-docs/assets/widgets/owl.jpg',
                fit: BoxFit.fill,
              ),
            ),

            const SizedBox(height: 30),

            const Text('Image.asset (Local File)'),
            // Make sure you have added assets/owl.jpg to your pubspec.yaml!
            SizedBox(
              width: 100,
              height: 100,
              child: Image.asset('assets/candy.png'),
            ),
          ],
        ),
      ),
    );
  }
}
